# BB84 Simulation

This package contains a simple educational simulation of the BB84 protocol.

## File
- `code/bb84_simulation.py`

## Features
- random Alice bits and bases
- random Bob bases
- sifted key extraction
- QBER calculation
- optional intercept-resend eavesdropping demo

## Run

```bash
pip install qiskit qiskit-aer
python code/bb84_simulation.py
```
