# BB84 Protocol

BB84 is a quantum key distribution protocol using non-orthogonal states.

## Bases
- Z basis: |0⟩, |1⟩
- X basis: |+⟩ = (|0⟩ + |1⟩)/√2, |−⟩ = (|0⟩ − |1⟩)/√2

## Protocol Steps
1. Alice chooses random bits and bases (Z or X), prepares corresponding states.
2. Bob measures each qubit in a random basis (Z or X).
3. Public discussion: Alice and Bob reveal bases (not outcomes) and discard mismatched cases.
4. The remaining bits form the sifted key.
5. Error rate estimation + (optionally) error correction and privacy amplification.

## Security Intuition
Non-orthogonal states cannot be perfectly distinguished. Any eavesdropping introduces errors detectable by Alice and Bob.

## Connection to Classical Cryptography
Unlike classical key exchange, BB84 provides information-theoretic security based on physical principles (no-cloning and measurement disturbance).
